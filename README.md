# DSSAT
Create weather file for DSSAT
This code generates .WTH weather file that is used for DSSAT simulation
requirments:
- Python > 3.0.0
- excel file (.xlsx) contains ['DATE',	'TMIN',	'TMAX',	'WIND',	'RAIN',	'SRAD'] columns
- Temperature in degree celsius
-  WIND in m/s
-  RAIN in mm
-  SRAD "Net downwardSolar radiation" in J/m^2
